# 📝 App de Tareas - PilarTecno

Proyecto desarrollado para **Pilar Tecno** y el **Polo Tecnológico de La Rioja**.

Esta aplicación permite registrar tareas y gestionar accesos mediante login. Está construida utilizando **HTML**, **CSS** y **JavaScript**.

---

## 🚀 Funcionalidades principales

- Registro y validación de usuario.
- Inicio de sesión.
- Carga y listado de tareas.
- Interfaz amigable y responsive.
